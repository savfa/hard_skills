# Практические проекты (Challenges)

Здесь вы будете реализовывать проекты для закрепления знаний. Каждый проект — это полноценный кейс.

### 1. Task Manager (Middle-)
*   **Задача:** Продвинутый список задач.
*   **Стек:** React + Redux Toolkit.
*   **Что реализовать:** CRUD (создание, чтение, обновление, удаление), фильтрация, LocalStorage, Undo/Redo действий через Redux.
*   **Поиск:** "Redux undo redo pattern", "React CRUD application best practices".

### 2. E-Commerce Catalog (Middle)
*   **Задача:** Каталог товаров.
*   **Стек:** RTK Query.
*   **Что реализовать:** Запросы к API (используйте JSONPlaceholder), пагинация, фильтрация товаров по категориям, корзина, оптимизация кэширования.
*   **Поиск:** "RTK Query pagination", "Shopping cart state management Redux".

### 3. Real-time Dashboard (Middle+)
*   **Задача:** Панель мониторинга.
*   **Стек:** WebSockets или SSE.
*   **Что реализовать:** Отображение данных в реальном времени, графики (используйте Recharts или Chart.js), обработка потока данных в Redux.
*   **Поиск:** "React real-time charts integration", "Redux handling high frequency updates".

### 4. Multi-step Form Builder (Senior-)
*   **Задача:** Сложный конструктор форм.
*   **Стек:** Zod, React Hook Form, FSD.
*   **Что реализовать:** Динамическая генерация полей, сложная валидация, сохранение прогресса, переходы между шагами.
*   **Поиск:** "React Hook Form multi-step tutorial", "Zod schema validation form".
