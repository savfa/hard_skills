# Практические проекты (Challenges)

Здесь вы будете реализовывать проекты для закрепления знаний.

1. [ ] **Task Manager:** Продвинутый список задач (CRUD, фильтрация, LocalStorage, Undo/Redo).
2. [ ] **E-Commerce Catalog:** RTK Query, корзина, пагинация, фильтры.
3. [ ] **Real-time Dashboard:** Работа с потоковыми данными, графики.
4. [ ] **Multi-step Form Builder:** Сложные формы, валидация (Zod), Redux.
