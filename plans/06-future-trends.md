# Future Frontend Trends & Horizons

Этот план поможет вам ориентироваться в современном ландшафте фронтенд-разработки и понимать, какие технологии стоит изучать для расширения кругозора ("T-shaped" developer).

## 1. Новые фреймворки и архитектурные подходы
- [ ] **Astro**: Изучение концепции "островов" (Islands Architecture) для быстрой загрузки контентных сайтов.
- [ ] **Qwik**: Понимание концепции "resumability" (возобновляемости) для достижения мгновенной интерактивности.
- [ ] **SolidJS**: Изучение реактивности без Virtual DOM.
- [ ] **Next.js (App Router)**: Глубокое погружение в Server Components и серверные архитектуры.

## 2. Инструментарий (Tooling)
- [ ] **Vite**: Понимание того, почему это стандарт и как он работает.
- [ ] **Rust-based Tooling**: Знакомство с Turbopack/Rspack как будущим сверхбыстрых сборщиков.
- [ ] **Bun**: Исследование возможностей Bun как рантайма и пакетного менеджера (замена Node.js).

## 3. Стилизация и UI
- [ ] **Tailwind CSS**: Внедрение в рабочий процесс.
- [ ] **Zero-runtime CSS-in-JS**: Использование библиотек типа Panda CSS или Vanilla Extract.
- [ ] **Headless UI (Radix, Ark UI)**: Изучение создания доступных (accessible) компонентов без привязки к стилям.

## 4. Backend-for-Frontend (BFF) и API
- [ ] **GraphQL**: Изучение основ запросов и схем.
- [ ] **tRPC**: Понимание безопасности типов (type-safety) в связке клиент-сервер.
- [ ] **Server Actions**: Работа с данными напрямую в React-компонентах.

## 5. AI в разработке
- [ ] **LLM Integration**: Интеграция AI (Vercel AI SDK) в пользовательские интерфейсы.
- [ ] **AI-assisted Development**: Практика использования AI для рефакторинга и написания тестов.

## 6. WebAssembly (Wasm)
- [ ] **Wasm Basics**: Понимание того, как и когда использовать Rust/Go для высокопроизводительных задач в браузере.
