# Architecture, Tooling & Testing

## Architecture
- [ ] Feature-Sliced Design (FSD): слои, слайсы, сегменты.
- [ ] Монорепозитории (pnpm workspaces / Turborepo).

## Tooling
- [ ] Vite: настройка сборки, плагины.
- [ ] Линтинг (ESLint, Prettier).
- [ ] Git Hooks (Husky, lint-staged).

## Testing
- [ ] Unit-тесты (Vitest / Jest).
- [ ] Тестирование компонентов и хуков (React Testing Library).
- [ ] Интеграционные тесты с MSW (Mock Service Worker).
- [ ] E2E-тестирование (Playwright).
