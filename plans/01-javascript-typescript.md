# JavaScript & TypeScript Roadmap

## JavaScript Core
- [ ] Event Loop: макро- и микрозадачи, приоритеты выполнения.
- [ ] Замыкания (Closures) и области видимости (Scope).
- [ ] Прототипы и наследование, классы.
- [ ] Управление памятью: сборщик мусора, утечки памяти, поиск через профилировщик.
- [ ] Асинхронное программирование: Promises, async/await, обработка ошибок.

## TypeScript Advanced
- [ ] Базовая типизация и интерфейсы.
- [ ] Generics: использование и ограничение.
- [ ] Utility Types (Partial, Pick, Omit, Record, Parameters, ReturnType, etc.).
- [ ] Conditional Types и `infer`.
- [ ] Type Guards (user-defined, typeof, instanceof).
- [ ] Конфигурация `tsconfig.json` (strict mode, paths, etc.).
