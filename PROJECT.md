# Сквозной проект: Task Manager Dashboard

Этот проект будет сопровождать вас на протяжении всего обучения. Вы будете переписывать и усложнять его на каждом этапе, применяя полученные знания.

## Описание
Task Manager Dashboard — приложение для управления задачами (CRUD).
Функционал:
- Создание, чтение, редактирование, удаление задач.
- Фильтрация и поиск задач.
- Статусы выполнения.

## Этапы развития

### Итерация 1: JS Core & TypeScript
- **Цель**: Реализовать бизнес-логику на чистом JS/TS (без фреймворков).
- **Задачи**:
    - Использовать модульную систему.
    - Реализовать менеджер состояния (на классах или объектах).
    - Добавить типизацию TypeScript.
    - Написать тесты для логики (Vitest).

### Итерация 2: React & Redux
- **Цель**: Перенос логики в React-приложение.
- **Задачи**:
    - Интеграция React.
    - Перенос состояния в Redux Toolkit.
    - Использование кастомных хуков.

### Итерация 3: Оптимизация и API
- **Цель**: Работа с сервером и оптимизация производительности.
- **Задачи**:
    - Подключение RTK Query для работы с API.
    - Использование MSW для моканья бэкенда.
    - Оптимизация рендеринга (memo, useCallback).
    - Внедрение Code Splitting (lazy loading компонентов).

### Итерация 4: Архитектура и Тестирование
- **Цель**: Масштабирование и промышленное качество.
- **Задачи**:
    - Рефакторинг под Feature-Sliced Design (FSD).
    - Добавление E2E-тестов (Playwright).
    - Настройка CI/CD (lint-staged, husky).
