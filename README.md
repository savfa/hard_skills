# Frontend Hard Skills Roadmap

Добро пожаловать в репозиторий для прокачки ваших фронтенд-навыков. Здесь вы найдете теоретические планы и практические задачи, которые помогут вам выйти на новый уровень владения стеком **React + Redux** и сопутствующими технологиями.

## Навигация по планам обучения

Каждый файл содержит чек-лист для отслеживания вашего прогресса:

- [JavaScript & TypeScript](plans/01-javascript-typescript.md)
- [React & Redux](plans/02-react-redux.md)
- [Browser, Performance & Security](plans/03-browser-performance-security.md)
- [Architecture, Tooling & Testing](plans/04-architecture-tooling-testing.md)
- [Практические проекты (Challenges)](plans/05-challenges-projects.md)

## Как пользоваться этим репозиторием
1. Выберите интересующую тему.
2. Изучите теорию, используя предложенные ресурсы.
3. Отмечайте выполненные пункты в чек-листах: `[x]`.
4. Реализуйте практические задачи в папке `/challenges`.
